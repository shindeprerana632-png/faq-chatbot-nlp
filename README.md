# FAQ Chatbot using NLP

This project is an AI-based FAQ chatbot that answers user queries using Natural Language Processing.

## Features
- Text preprocessing using NLTK
- TF-IDF Vectorization
- Cosine Similarity matching
- Flask Web Interface

## How it works
The chatbot compares the user's question with stored FAQs and returns the most relevant answer.

## Run Locally

pip install -r requirements.txt
python app.py

Open http://127.0.0.1:5000
